# import-export-sql-database
Import and Export SQL databases

A quick and easy way to export/import your sql database, with an easy visual basic programm.

For this project i have used the following dll https://www.codeproject.com/Articles/256466/MySqlBackup-NET-MySQL-Backup-Solution-for-Csharp-V, which is located in https://github.com/MySqlBackupNET/MySqlBackup.Net

Just type to the appropriate boxes your server name, user name, password and the name of the database that you want to extract/import and hit Import or Export Database buttons accordingly.
